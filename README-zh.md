# BPDF — AutoCAD 批次出圖工具

**一個指令，將模型空間內所有圖框批次輸出為獨立 PDF。**

建築師做的工具，給建築師用。不用再一張一張手動出圖。

---

## 功能特色

- 點選圖框自動偵測 Block 名稱，不需要手動輸入
- 可選擇只出這一張，或出所有相同名稱的圖框
- 輸出順序：由上到下、由左到右自動排序
- 動態讀取電腦內的出圖樣式表和圖紙尺寸
- 記憶上次設定，下次直接 Enter 沿用
- 出圖前顯示確認畫面
- 相容 AutoCAD 2014 以上版本

---

## 使用示範

```
指令: BPDF

點選圖框 Block（或按 Enter 手動輸入名稱）:
→ [點選圖框]

偵測到 Block: A$C44896E9B
出圖模式 [1=只出這張 / A=出所有相同圖框] <A>: A

PDF 檔名前綴 <frame>: 中壢案-建照
輸出資料夾 <C:\Desktop>:
出圖樣式: （從清單選擇）
圖紙尺寸: （從清單選擇）
比例分母: 100  ← 即 1:100

找到 26 個圖框，開始輸出...
  [1/26] 完成
  [2/26] 完成
  ...
  [26/26] 完成 ✓
```

26 張圖不到一分鐘搞定。

## 安裝方式

1. 下載 `batch-pdf.lsp`
2. 在 AutoCAD 命令列輸入 `APPLOAD`
3. 點「啟動組」→「內容」→「加入」
4. 選擇 `batch-pdf.lsp` → 確定

設定完成後，每次開啟 AutoCAD 都會自動載入，直接輸入 `BPDF` 即可使用。

**建議存放位置：**
```
C:\Users\你的名字\AppData\Roaming\Autodesk\AutoCAD 2022\R24.1\cht\Support
```

不確定路徑？在 AutoCAD 輸入 `(findfile "acad.lsp")` 查詢。

## 使用步驟

**Step 1 — 執行指令**
```
BPDF
```

**Step 2 — 點選圖框**

直接在模型空間點選任一圖框，程式自動讀取 Block 名稱。

或按 Enter 手動輸入 Block 名稱。
（不知道名稱？點選圖框後輸入 `LIST` 查詢 `圖塊名稱` 那行）

**Step 3 — 選擇出圖範圍**

```
[1=只出這張 / A=出所有相同圖框] <A>:
```

**Step 4 — 依序設定**

| 設定項目 | 說明 |
|---------|------|
| PDF 檔名前綴 | 例如輸入「中壢案」，輸出為「中壢案_1.pdf」|
| 輸出資料夾 | 直接 Enter = 桌面 |
| 出圖樣式表 | 輸入數字選擇 |
| 圖紙尺寸 | 輸入數字選擇 |
| 出圖比例 | 例如 100 = 1:100，輸入 0 或 F = 佮滿，Enter = 沿用上次設定 |

**Step 5 — 確認後自動出圖**

確認畫面按 OK → 輸入 `Y` → 完成！

## 為什麼要用 BPDF？

AutoCAD 內建的批次出圖（Publish）是針對 Layout 空間設計的。

BPDF 專為**模型空間（Model Space）工作流程**設計——台灣及亞洲許多事務所習慣將所有圖框排列在同一個模型空間內，官方的 Publish 工具在這種情況下幫不上忙。

類似工具在華語圈有「秋楓批量打印」等，但多為封閉原始碼且僅支援中文介面。BPDF 開源、英文介面，全球用戶都能使用。
