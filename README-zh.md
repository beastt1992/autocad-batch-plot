# BPDF — AutoCAD 批次出圖工具

**一個指令，將模型空間內所有圖框批次輸出為獨立 PDF。**

給 AutoCAD 使用者的工具。不用再一張一張手動出圖。

---

## 功能特色

- 簡單的對話框介面，填好設定按 OK 就出圖
- 點選「Pick」直接點圖框自動偵測 Block 名稱
- 可選擇只出這一張，或出所有相同名稱的圖框
- 輸出順序：由上到下、由左到右自動排序
- 動態讀取電腦內的出圖樣式表和圖紙尺寸
- 自動記憶上次設定
- 出圖前顯示確認畫面
- 相容 AutoCAD 2014 以上版本

## 安裝方式

1. 下載 `batch-pdf.lsp`
2. 在 AutoCAD 命令列輸入 `APPLOAD`
3. 點「啟動組」→「內容」→「加入」
4. 選擇 `batch-pdf.lsp` → 確定

設定完成後，每次開啟 AutoCAD 都會自動載入，直接輸入 `BPDF` 即可使用。

**建議存放位置：**
```
C:\Users\你的名字\AppData\Roaming\Autodesk\AutoCAD 2022\R24.1\cht\Support
```

不確定路徑？在 AutoCAD 輸入 `(findfile "acad.lsp")` 查詢。

## 使用步驟

**Step 1 — 執行指令**
```
BPDF
```

**Step 2 — 填寫對話框**

| 欄位 | 說明 |
|------|------|
| Block Name | 輸入 Block 名稱，或點「Pick」直接點圖框自動偵測 |
| PDF Prefix | 檔名前綴，例如「中壢案」→ 輸出「中壢案_1.pdf」|
| Output Folder | 輸出資料夾路徑，預設桌面 |
| 出圖樣式 | 從清單選擇 |
| 圖紙尺寸 | 從清單選擇（A0–A4）|
| 比例 | `100` = 1:100，`0` 或 `F` = 佮滿，Enter = 沿用上次 |
| 出圖模式 | 出所有相同圖框，或只出這一張 |

**Step 3 — 按 OK 確認出圖**

確認畫面輸入 `Y` → 完成！

## 為什麼要用 BPDF？

AutoCAD 內建的批次出圖（Publish）是針對 Layout 空間設計的。

BPDF 專為**模型空間（Model Space）工作流程**設計——台灣及亞洲許多事務所習慣將所有圖框排列在同一個模型空間內，官方的 Publish 工具在這種情況下幫不上忙。

類似工具在華語圈有「秋楓批量打印」等，但多為封閉原始碼且僅支援中文介面。BPDF 開源、英文介面，全球用戶都能使用。
